# screenshots/

Boot/attract-screen captures of the locally-staged commercial ROM corpus, the RustySNES port of
RustyNES's screenshot pipeline. Generated by `scripts/screenshots/generate.sh` (which runs the
`commercial_screenshots` harness + ImageMagick).

- `external/<chip>/` — Core/Curated boards (None, DSP-1, GSU-1/2, SA-1).
- `besteffort/<chip>/` — BestEffort boards (CX4, S-DD1, SPC7110, OBC1, ST010, DSP-2/4, ...).
- `montage.png` — showcase grid.

Only these PNGs are committed (derived emulator output); the ROMs + coprocessor firmware that
produced them stay gitignored under `tests/roms/external/` (ADR 0003 — never commit ROM bytes).
Regenerate with `scripts/screenshots/generate.sh [frames]`.
