# Fullsnes — Nocash SNES Specs (RustySNES Reference Mirror)

> **Reformatted mirror for RustySNES development reference.** This is nocash's *Fullsnes — Nocash SNES Specs*, converted from the [official HTML](https://problemkaputt.de/fullsnes.htm) (fetched 2026-07-22) into navigable Markdown. Content is reproduced faithfully; only structure (headings, a table of contents, fenced code blocks, cross-file links) was added. Register/timing tables are preserved verbatim as `text` code fences so that column alignment, hex notation, and bit-field layouts are never mangled.

See [About / Credits](99-about-and-index.md#aboutcredits) for the full attribution. Updates: <https://problemkaputt.de/fullsnes.htm> (HTML) and <https://problemkaputt.de/fullsnes.txt> (text).
>
> Additions made for this mirror are always fenced as `> **Note (RustySNES ref):**` blockquotes and never alter nocash's text.

## Why this is split into multiple files

The original is one flat ~1.5 MB HTML page. Converted to Markdown it exceeds 400 KB, so it is split into logical section files (below). This index is the linked master table of contents. Every heading carries an HTML `id` matching nocash's original anchor name, so old `#anchor` cross-references resolve, and a GitHub-style slug for normal Markdown linking.
